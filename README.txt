Space Marine

This project is a canvas game remember "Space Invader" developed with Javascript.
There will be some goals to meet:

 - The game must be able to play on any browser
 - It must be coded in ES6.

Get started

 - We have developed all the necessary class to allow customization of the game
 - We did not have time to create a real customization of the game for the player. But we want to continue development and finish the game.

Prerequisites

 - Internet

Deployment

 - Open the html page of project
